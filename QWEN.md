# Breakout LB426 — Проектная документация

## Обзор проекта

**Breakout LB426** — классическая аркадная игра Breakout, реализованная на **Rust** с использованием игрового движка **Bevy 0.18.1**.

### Основная концепция
Игрок управляет платформой (paddle) внизу экрана, которая должна отбивать мяч. Мяч рикошетит от стен и разрушает блоки в верхней части экрана. Цель — разбить все блоки, не дав мячу упасть вниз.

### Технологии
- **Язык:** Rust (edition 2021)
- **Движок:** Bevy 0.18.1 (ECS-архитектура, 2D-рендеринг)
- **Аудио:** MP3/OGG файлы для звуковых эффектов

## Архитектура

Проект использует **ECS-архитектуру** (Entity-Component-System) движка Bevy:

### Компоненты (Components)

| Компонент | Описание |
|-----------|----------|
| `Paddle` | Платформа игрока со скоростью и направлением |
| `Ball` | Мяч с вектором скорости |
| `Velocity` | Общий компонент скорости |
| `Collider` | Компонент для обработки столкновений |

### Системы (Systems)

| Система | Описание |
|---------|----------|
| `setup_camera` | Инициализация 2D-камеры |
| `setup_level` | Генерация сетки блоков (5 рядов × 10 колонок) |
| `setup_paddle` | Создание платформы игрока |
| `setup_ball` | Создание мяча |
| `paddle_movement` | Обработка движения платформы |
| `ball_movement` | Физика мяча, столкновения со стенами и платформой |

### Игровое поле

| Параметр | Значение |
|----------|----------|
| Размеры поля | 960×540 (координаты: x ∈ [-480, 480], y ∈ [-300, 270]) |
| Платформа | 120×20, позиция y = -250 |
| Мяч | 20×20 |
| Блоки | 50×20, начинаются с y = 250 |
| Расстояние между блоками | 5 пикселей |

## Сборка и запуск

### Требования

- **Rust:** 1.75+ (рекомендуется последняя стабильная версия)
- **Cargo:** Менеджер пакетов Rust

### Команды

```bash
# Запуск игры
cargo run

# Запуск в режиме отладки
cargo run --release

# Проверка кода
cargo check

# Сборка проекта
cargo build

# Сборка релизной версии
cargo build --release

# Запуск тестов (если есть)
cargo test

# Форматирование кода
cargo fmt

# Линтинг кода
cargo clippy
```

### Отладка в VS Code

Проект включает конфигурацию для отладки через `.vscode/launch.json`:

| Конфигурация | Описание |
|--------------|----------|
| `Debug executable 'breakout_lb426'` | Запуск в режиме отладки |
| `Debug unit tests in executable 'breakout_lb426'` | Отладка тестов |

## Особенности реализации

### Физика мяча

- Отскок от левой/правой стены (x = ±480)
- Отскок от верхней стены (y = 270)
- Отскок от платформы при пересечении
- Респаун мяча при падении вниз (y < -300)

### Генерация уровня

- 5 рядов блоков
- 10 колонок блоков
- Цвет блоков зависит от ряда (градиент зелёного)
- Расстояние между блоками: 5 пикселей

## Примечания для разработчиков

1. **Модульная структура:** Код разделён на логические модули для удобства поддержки
2. **ECS-паттерн:** Все игровые объекты — сущности с компонентами, логика — в системах
3. **Bevy 0.18:** Используется новая версия API с `Camera2d`, `ButtonInput`, `Res<Time>`
